# taa2026-higgs-boson

Para iniciar todo adecuadamente:
- Necesitan tener "unzip" en su terminal
- Ejecutar en terminal "chmod +x setup.sh"
- Agregar sus credenciales de kaggle en el entorno de trabajo, esto es con:
    export KAGGLE_USERNAME=<nombre_usuario>
    export KAGGLE_KEY=<api_key>
    
    *(Guardarse o anotarse api_key de kaggle en algun lado por las dudas, creo que hay que setearla cada vez que reinician entorno de trabajo)*
- Ejecutar el shell script anterior ("./setup.sh")
