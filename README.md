# taa2026-higgs-boson

Para iniciar todo adecuadamente:
- Necesitan tener "unzip" en su terminal
- Ejecutar en terminal "chmod +x setup.sh"
- Agregar sus credenciales de kaggle a Github Codespaces Secrets, esto es:
    En settings del repo -> Secrets and Variables -> Codespaces
    Ahi poner estos dos mapeos de <clave, valor>
    KAGGLE_USERNAME -> <nombre_usuario>
    KAGGLE_API_TOKEN -> <api_key>
    
    *(Guardarse o anotarse api_key de kaggle en algun lado por las dudas)*
- Ejecutar el shell script anterior ("./setup.sh")
